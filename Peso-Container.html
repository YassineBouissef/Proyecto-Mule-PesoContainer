1- Descripción del proyecto

Peso-Container es una aplicación que informa a tiempo real el peso de un contenedor de ropa, para no sobrepasar el limite su capacidad. 

2- ThingSpeak y Simulación

Creamos un canal en ThingSpeak con un campo, el peso. Luego escribimos un código en Java para incrementar el peso. Compilamos y ejecutamos el código.

 

3-Anypoint Studio

Creamos un proyecto en Anypoint para la obtención de datos del archivo .json que genera nuestro canal de ThingSpeak, que creamos antes, para generar un Map que contega datos y los muestre.

La implementación:

  1. Componente Poll: Manda correos cada tiempo determinado.
  2. Conector HTTP: indica la fuente de datos que se usa.
  3. Componente Java: procesa el formato del json a cadena.
  4. SMTP: Manda el objeto al correo indicado.
